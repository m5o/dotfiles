This file skeleton is intended to work with Mr Bones
(http://www.viget.com/extend/creating-gems-with-mr-bones/).
Put it in your home directory under .mrbones.

To get the most use out of it, you should set the following
environment variables.

MY_NAME
MY_EMAIL
MY_GITHUB_LOGIN
MY_ORG - optional; specifies that you are part of a larger group
